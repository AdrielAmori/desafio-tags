Introdução ao HTML

O HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para desenvolver páginas na Web. Ela atua como o esqueleto de uma página web, dando estrutura aos textos, imagens e outros conteúdos. HTML não é uma linguagem de programação, mas uma linguagem de estruturação de conteúdo para a web.

Estrutura básica de uma página HTML

Uma página HTML padrão tem uma estrutura específica, incluindo elementos como DOCTYPE, html, head e body. Dentro da head estão meta-informações, como o título da página, enquanto o corpo contém o conteúdo principal, que é visível para os usuários.

"HTML é a espinha dorsal de qualquer página web. É o que fornece a estrutura e a organizidade, tornando a web acessível e útil para todos." - Autor desconhecido

Tags HTML

As tags HTML especificam como o conteúdo será formatado. Por exemplo, a tag < p > cria um parágrafo, enquanto a tag < img > insere uma imagem. Cada tag serve a um propósito específico e controla como o conteúdo é exibido e comportado. Há também tags para subscritos e sobrescritos, ideais para uso em equações matemáticas ou notas de rodapé.

Por exemplo, a tag < sub > diminui a linha de base do texto, criando um efeito de texto subscrito, assim: H2O

Enquanto isso, a tag < sup > aumenta a linha de base do texto, criando um efeito de texto sobrescrito, como neste exemplo: E = mc2

Atributos HTML

Os atributos HTML são usados para fornecer informações adicionais sobre elementos HTML, como especificar a fonte de uma imagem com o atributo src em uma tag < img > ou adicionar um id ou class a uma tag para estilização com CSS.

Formatação de texto em HTML

Ao usar tags HTML, o texto pode ser formatado de várias maneiras, como em negrito < b >, itálico < i >, sublinhado < u >, etc. Outros elementos, como < h1 > a < h6 >, especificam os cabeçalhos.

Listas em HTML

HTML permite criar listas ordenadas (numeradas) e não ordenadas (com marcadores) com as tags < ol > e < ul >, respectivamente. Para os itens da lista, usamos a tag < li >.

Exemplos de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplos de lista não ordenada:

  • Casa
  • Carro
  • Bola

Imagens e Links

As imagens são incorporadas em páginas web usando a tag < imgb > com o atributo src, enquanto os links são criados com a tag e o atributo href, que leva a outros documentos ou páginas da web.

Boas práticas em HTML

As boas práticas incluem manter o código limpo e organizado, fechando todas as tags, usando atributos alt para todos os elementos de imagem, usando cabeçalhos corretamente para organização, validando seu código e evitando tecnologias obsoletas ou descontinuadas.
